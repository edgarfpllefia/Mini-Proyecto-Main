# Nombre del proyecto

Incidencias JS

Ejercicio 5 de JS de AstroDaw para la gestión de incidencias a traves de formularios con JS.

# Descripción general

Tabla de incidencias, creadas desde un Array. Es posible crear, eliminar o editar desde diferentes botones y que hagan sus funciones a traves de index.js

# Tecnologías utilizadas

-HTML5
-CSS3/Boostrap
-JavaScript

# Estructura del proyecto

El proyecto está estructurado en 6 archivos js. 

- index.js
- data.js
- dom.js
- function/form.js
- function/table.js
- event/form.js
